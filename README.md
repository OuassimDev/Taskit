# <img width="30" height="30" alt="image" src="checklist.png" /> Taskit

Taskit is a simple, elegant, and minimalistic To-Do List web application built using vanilla frontend web technologies[cite: 5]. It now features complete user authentication and real-time cloud data persistence.

---

## 🌐 Live Demo

Check out the live version here: [https://taskit-do.vercel.app/](https://taskit-do.vercel.app/)[cite: 5]

---

## ✨ Features

- **User Authentication:** Securely sign up or sign in to manage your personal tasks[cite: 1, 2].
- **Task Management:** Create, edit, and delete tasks seamlessly[cite: 2, 5].
- **Task Statuses:** Mark tasks as completed with a visual cross-out animation[cite: 2, 3].
- **Deadlines:** Assign specific completion dates to your tasks with an embedded calendar picker[cite: 2].
- **Theme Toggling:** Switch between light mode and dark mode layouts based on your preference[cite: 1, 2].
- **Responsive Design:** Fully optimized with a fluid interface for both desktop and mobile layouts[cite: 3, 5].

---

## 🛠️ Technologies Used

- **HTML:** Semantic structure[cite: 1, 5].
- **CSS:** Custom properties, CSS Grid/Flexbox, animations, and dark mode styling[cite: 3, 5].
- **JavaScript:** Vanilla asynchronous DOM manipulation and event-driven architecture[cite: 2, 5].
- **Supabase:** Backend-as-a-Service handling User Authentication and Postgres Database storage[cite: 1, 2, 4, 5].
